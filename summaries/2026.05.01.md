[URL](https://github.com/NousResearch/hermes-agent) / [Homepage](https://hermes-agent.nousresearch.com) / 言語: Python

Hermes Agent は Nous Research が開発した自己改善型 AI エージェントです。内蔵の学習ループでスキルを経験から生成・改善し、会話履歴を検索して利用者モデルを継続的に深めます。Telegram や Discord, Slack, WhatsApp, Signal, CLI など複数のクライアントに対応し、任意のモデル（ OpenAI, Hugging Face, NVIDIA NIM など）を切り替えて利用できます。低コストのクラウドや VPS で常時稼働可能で、TUI ベースのリッチな操作性や音声メモ転写、永続的学習機能を備えています。


## `QuipNetwork/hashsigs-solidity`

> Solidity contracts implementing hash based post quantum signatures

[URL](https://github.com/QuipNetwork/hashsigs-solidity) / 言語: Solidity

このリポジトリは、Winternitz One-Time Signature Plus ( WOTS+ ) のハッシュベースのポスト量子署名を Solidity で実装したものです。開発には Hardhat と Foundry を併用し、Foundry のインストールと Node.js 依存関係のセットアップ、テスト実行 ( forge test )、およびビルド ( forge build ) の手順が記載されています。ライセンスは GNU Affero GPLv3 です。


## `QuipNetwork/ethereum-sdk`

> 

[URL](https://github.com/QuipNetwork/ethereum-sdk) / 言語: TypeScript

Quip Network SDK は、Quip Network の EVM スマートコントラクトとそれを操作する TypeScript SDK を含むリポジトリです。開発はアクティブで本番利用前の段階です。Node.js を前提に、環境変数を .env で設定し、Hardhat を使ったコントラクトのテストやローカルノード起動が可能です。ネットワーク追加やデプロイ用のスクリプトがあり、権限のある担当者のみ実行する必要があります。SDK のビルドは bun、テストは bun run test、公開は npm publish を使用します。


## `openai/codex-plugin-cc`

> Use Codex from Claude Code to review code or delegate tasks.

[URL](https://github.com/openai/codex-plugin-cc) / 言語: JavaScript

Codex plugin for Claude Code は、Claude Code ワークフロー内から Codex を使ってコードレビューやタスク委任を行えるプラグインです。インストール手順、必要条件（ChatGPT サブスクリプションまたは OpenAI API key、Node.js 18.18 以降）、セットアップ方法、主要なスラッシュコマンド（/codex:review、/codex:adversarial-review、/codex:rescue など）と使い方が説明されています。背景実行やブランチ比較、Codex のインストール・ログイン手順もカバーしており、Claude Code ユーザーが簡単に Codex を統合できるよう設計されています。
